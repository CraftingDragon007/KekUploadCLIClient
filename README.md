# KekUploadCLIClient

This is a CLI client for [UploadServer](https://oss.kotw.dev/uploadserver) written in C#.

<br>

## Prerequirements

- dotnet

<br>

## Building

```sh
cd KekUploadCLIClient
dotnet build -c release
```

The executable file and its libraries will be located in `KekUploadCLIClient/bin/release/net<version>/`
